✨ Overview
Applying to internships can get messy with multiple deadlines and statuses.  
**InternHub** simplifies this by providing a centralized dashboard to track everything in one place.

🎯 Features
- 📊 Track applications, interviews, offers, and rejections  
- ⏳ Manage upcoming deadlines  
- 🔄 Monitor recent activity  
- 📈 View ATS match scores  
- 🗂 Multiple views: Kanban, List, Timeline  
- 🤖 AI tools: JD Analyzer & Cover Letter  

🛠 Tech Stack
HTML, CSS, JavaScript  

💡 Purpose
Helps students stay organized and manage internship applications efficiently.

🚀 Future Scope
Notifications, resume insights, analytics, mobile support
